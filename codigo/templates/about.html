Guia de Uso

Páginas do Website

1. Página "Início"

A página "Início" proporciona uma visão geral do projeto, apresentando informações sobre seu objetivo, as tecnologias empregadas e os membros da equipe envolvida. Ela funciona como uma introdução ao site e à sua proposta de promover a transparência nos gastos públicos do Distrito Federal.

2. Página "Análise de Gastos"

A página "Análise de Gastos" apresenta uma análise detalhada dos recursos financeiros alocados pelo Governo do Distrito Federal em diversas áreas, como saúde, educação e segurança. Os dados são exibidos de forma clara por meio de gráficos e tabelas.

2.1 Gráficos Interativos

Os gráficos ilustram a distribuição dos gastos em diferentes áreas. Para um entendimento mais aprofundado:

  • Ao posicionar o cursor sobre qualquer parte do gráfico, serão exibidos detalhes adicionais, como o ano e o valor, facilitando a compreensão das informações apresentadas.

Abaixo, é apresentada uma imagem exemplificando os detalhes do gráfico de "Gastos com Compras" no mês de abril de 2024.

Baixar os Dados

Caso deseje armazenar as informações para consulta futura, é possível realizar o download dos gráficos e tabelas em formato PDF. Abaixo, apresentamos um exemplo de como efetuar o download:

  • Exemplo de Download dos Gráficos:
    1. A primeira seta na imagem indica a opção para o download geral dos gráficos, permitindo a obtenção de todos os gráficos simultaneamente.
    2. A segunda seta representa a opção de download individual, permitindo que cada gráfico seja baixado separadamente.

2.2 Tabelas

Na seção de tabelas, é possível selecionar o tipo de gasto a ser analisado por meio do botão "Selecionar Gasto". As opções disponíveis são:

  • Compras: Esta seção é exibida automaticamente ao acessar a área de tabelas. Nela, é possível selecionar o mês desejado para consultar os dados correspondentes. As tabelas de compras apresentam informações detalhadas, como:
    • Empresa
    • CNPJ
    • Ano
    • Valor
    • Objeto
  • Despesas: Esta opção está subdividida em várias subseções, como Ministério da Saúde, Ministério do Trabalho e outras áreas relevantes. O usuário pode selecionar o ano de interesse e explorar as informações de cada subseção. As tabelas de despesas incluem dados como:
    • Ano
    • Órgão
    • Código do Órgão
    • Empenhado
    • Líquido
    • Pago

Filtragem de Dados

A pesquisa pode ser realizada de maneira simples, conforme os seguintes passos:

  1. Na seção de compras, selecione o mês desejado.
  2. Na seção de despesas, selecione o ano desejado.
  3. Informe o critério de pesquisa na caixa apropriada e clique no botão "Pesquisar" para filtrar os dados.

Os resultados serão exibidos com base nos filtros aplicados, mostrando a quantidade de registros encontrados ao final da tabela.

Baixar PDF

Para realizar o download dos dados em formato PDF, siga os passos abaixo:

  1. Escolha o tipo de gasto. (opcional)
  2. Selecione o período desejado. (opcional)
  3. Aplique os filtros conforme necessário. (opcional)
  4. Clique no botão "Baixar Tabelas (PDF)" para salvar os dados em seu dispositivo.

Esta funcionalidade permite visualizar e armazenar exatamente os dados desejados, com opções de detalhamento claramente organizadas, conforme ilustrado na imagem abaixo.
